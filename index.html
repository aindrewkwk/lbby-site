Getting Started

From download to your first server in minutes.

Quick Start

  1. Download and install Lbby for your platform
  2. Open Lbby — the setup wizard will guide you
  3. Pick a server type: Paper (recommended for most users), Fabric, Forge, etc.
  4. Choose a Minecraft version and loader/build version if needed
  5. Set your server name, folder, RAM, and max players
  6. Click Install Server
  7. Open the Dashboard and click Start
  8. Open the Tunnel tab and start playit.gg
  9. Share the public tunnel address with friends

That's it: install → start → tunnel → share.

Server Types

Lbby supports 11 server types. Here's a quick guide:

  • Paper — Optimized server with plugin support. Best for most servers.
  • Fabric — Lightweight modern mod loader. Great for performance mods.
  • Forge — Classic mod loader with the largest mod ecosystem.
  • Purpur — Feature-rich Paper fork with extra customization options.
  • Folia — Multithreaded Paper fork for massive servers (regionalized).
  • NeoForge — Modern fork of Forge with active development.
  • Bukkit / Spigot — Original plugin API servers (compiled via BuildTools).
  • SpongeVanilla / SpongeForge — Standalone Sponge API servers.
  • Vanilla — Pure Minecraft, no mods or plugins.

Player Management

Manage players directly from the app:

  • View online players with Minecraft head avatars
  • OP / de-OP players
  • Kick, ban, and unban players
  • Teleport one player to another

Mods & Modpacks

The Mods page has four sections:

  • Modpacks — Install Modrinth .mrpack, paste a Modrinth link, or upload a CurseForge .zip
  • Browse Mods — Search Modrinth and install compatible server mods directly
  • Installed & Updates — View installed .jar files, check for updates, and update safely
  • Resource Packs — Add, list, remove, and open the server resource pack folder

Backups

  • Create manual ZIP backups of your server
  • Restore from any previous backup
  • Schedule automatic backups (Novice tier and above)
  • Old mod files are backed up before updates in .lbby-backups

Performance Tips

  • Use Settings → Performance → Low CPU if CPU spikes
  • Keep Optimized JVM flags enabled
  • Lower View Distance to 6–8
  • Lower Simulation Distance to 4–6
  • Use Pre-generate Chunks before players explore far from spawn

playit.gg Troubleshooting

If friends can't join, check these:

  • Dashboard shows the Minecraft server is Online
  • Tunnel tab shows playit.gg is running
  • Tunnel type is Minecraft Java / TCP
  • Local tunnel port matches server-port in server.properties (usually 25565)
  • Use the exact address shown by Lbby, including the port if shown
