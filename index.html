Quick Start

  1. Download and install Lbby for your platform
  2. Open Lbby — the setup wizard will guide you
  3. Choose a supported game
  4. For now, select Minecraft Java
  5. Pick a Minecraft server type: Paper recommended, Fabric, Forge, etc.
  6. Choose a Minecraft version and loader/build version if needed
  7. Set your server name, folder, RAM, and max players
  8. Click Install Server
  9. Open the Dashboard and click Start
  10. Open the Tunnel tab and start playit.gg
  11. Share the public tunnel address with friends

Minecraft Java

Minecraft Java is Lbby's first fully supported game. It includes guided setup, server types, mods, plugins, modpacks, player management, backups, tunnels, and performance tools.

Server Types

Lbby supports 11 Minecraft Java server types:

  • Vanilla — Pure Minecraft, no mods or plugins.
  • Paper — Optimized server with plugin support. Best for most servers.
  • Fabric — Lightweight modern mod loader. Great for performance mods.
  • Forge — Classic mod loader with the largest mod ecosystem.
  • Purpur — Feature-rich Paper fork with extra customization.
  • Folia — Multithreaded Paper fork for massive servers.
  • NeoForge — Modern fork of Forge.
  • Bukkit / Spigot — Original plugin API servers.
  • SpongeVanilla / SpongeForge — Sponge API servers.

Player Management

For Minecraft Java servers, click any online player to open the detail modal:

  • Stats — Health, food, XP bar, gamemode, position, active effects
  • Inventory — View items, give items, remove items, ender chest
  • Actions — Kill, heal, feed, freeze, spectate, gamemode, clear effects, set XP
  • OP/de-OP, kick, ban/unban, and teleport from the player list

Mods & Plugins

For Minecraft Java:

  • Modpacks — Install Modrinth .mrpack, paste a link, or upload CurseForge .zip
  • Browse Mods — Search Modrinth and install directly
  • Updates — Check for mod updates, update safely
  • Resource Packs — Add, list, remove server resource packs

Backups

  • Create manual ZIP backups of your server
  • Restore from any previous backup
  • Schedule automatic backups

Performance Tips

For Minecraft Java:

  • Use Settings → Performance → Low CPU if CPU spikes
  • Keep Optimized JVM flags enabled
  • Lower View Distance to 6–8
  • Use Pre-generate Chunks before players explore far

playit.gg Troubleshooting

  • Dashboard shows the server is Online
  • Tunnel tab shows playit.gg is running
  • Local port matches the server port, usually 25565 for Minecraft Java
  • Use the exact address shown by Lbby, including port if shown

License

How Licensing Works

Lbby uses Ed25519-signed JWT tokens for license activation. Keys are generated and verified entirely client-side — no backend required.

01

Ed25519 Signing

Licenses are signed with Ed25519 elliptic curve keys. The private key never leaves the generator; the public key is embedded in the app for verification.

02

JWT Tokens

Each license is a standard JSON Web Token containing user ID, tier, expiry, and optional device binding. Tokens are compact and easy to share.

03

Novice & Master Tiers

Two license tiers control feature access. Novice covers basic hosting; Master unlocks advanced tools and unlimited servers.

04

Device Binding

Licenses can optionally be bound to a device fingerprint, preventing the same key from being used on multiple machines.

05

Expiry Control

Set license validity from 30 minutes to 10 years. Tokens include a built-in expiry timestamp — the app enforces it automatically.
