From download to your first server in minutes.

Quick Start

  1. Download and install Lbby for your platform
  2. Open Lbby — the setup wizard will guide you
  3. Pick a server type: Paper (recommended), Fabric, Forge, etc.
  4. Choose a Minecraft version and loader/build version if needed
  5. Set your server name, folder, RAM, and max players
  6. Click Install Server
  7. Open the Dashboard and click Start
  8. Open the Tunnel tab and start playit.gg
  9. Share the public tunnel address with friends

That's it: install → start → tunnel → share.

Server Types

Lbby supports 11 server types:

  • Paper — Optimized server with plugin support. Best for most servers.
  • Fabric — Lightweight modern mod loader. Great for performance mods.
  • Forge — Classic mod loader with the largest mod ecosystem.
  • Purpur — Feature-rich Paper fork with extra customization.
  • Folia — Multithreaded Paper fork for massive servers.
  • NeoForge — Modern fork of Forge.
  • Bukkit / Spigot — Original plugin API servers.
  • SpongeVanilla / SpongeForge — Sponge API servers.
  • Vanilla — Pure Minecraft, no mods or plugins.

Player Management

Click any online player to open the detail modal:

  • Stats — Health, food, XP bars, gamemode, position, active effects
  • Inventory — View items, give items, remove items, ender chest
  • Actions — Kill, heal, feed, freeze, spectate, gamemode, clear effects, set XP
  • OP/de-OP, kick, ban/unban, and teleport from the player list

Mods & Plugins

  • Modpacks — Install Modrinth .mrpack, paste a link, or upload CurseForge .zip
  • Browse Mods — Search Modrinth and install directly
  • Updates — Check for mod updates, update safely
  • Resource Packs — Add, list, remove server resource packs

Backups

  • Create manual ZIP backups of your server
  • Restore from any previous backup
  • Schedule automatic backups

Performance Tips

  • Use Settings → Performance → Low CPU if CPU spikes
  • Keep Optimized JVM flags enabled
  • Lower View Distance to 6–8
  • Use Pre-generate Chunks before players explore far

playit.gg Troubleshooting

  • Dashboard shows the server is Online
  • Tunnel tab shows playit.gg is running
  • Local port matches server-port in server.properties (usually 25565)
  • Use the exact address shown by Lbby, including port if shown
